A polished portfolio powered by **React + Vite + Bootstrap 5.3** with dark/light theme toggle, glass cards, and an inline **PDF resume viewer** with **download**.

## Quick start
```bash
npm i
npm start
# open http://localhost:5173
```

## Replace the resume
- Put your resume at `public/resume.pdf` (replace the placeholder).
- The Resume section will automatically load it.

## Build & preview
```bash
npm run build
npm run preview
```

## Deploy
- **GitHub Pages**: use a Pages workflow or deploy the `dist` folder.
- **Netlify**: build command `npm run build`, publish directory `dist`.
- **Vercel**: import the repo; it auto-detects Vite.

## Edit content
- Update `src/data/profile.json` to change details.

## Stack
- React 18, Vite 5, Bootstrap 5.3 (CDN), Bootstrap Icons
